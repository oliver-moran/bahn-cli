bahn-cli
========

Node CLI to install and run a [bahn](https://github.com/oliver-moran/bahn) server.

To install: `npm install bahn -g`

To install and run a server in a (preferably empty) directory. Examples:

    bahn --create ~/bahn
    bahn --port 3000 ~/bahn
    bahn --forever

See:

    bahn --help
